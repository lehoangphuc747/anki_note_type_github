## 🎨 Tính năng

### Card 1 & Card 4 - Typing với màu sắc đậm
- **typeGood**: Màu xanh đậm (#00796B) với background và border
- **typeBad**: Màu đỏ đậm (#C62828) với background, border và **underline**
- **typeMissed**: Màu xanh dương đậm (#1E88E5) với background và border
- Hỗ trợ Night Mode

### Card 3 - Immersive View
- Giao diện tập trung, toàn màn hình
- Chỉ hiển thị câu ví dụ và nghĩa
- Background: #F8F9FA

### Card 4 - Review Container
- Khung trắng với border và shadow
- Hiển thị kết quả typing và câu ví dụ gốc

## 📝 Cách sử dụng

1. Mở Anki → **Tools** → **Manage Note Types**
2. Chọn note type của bạn hoặc tạo mới
3. Copy nội dung từ các file template tương ứng vào **Front Template** và **Back Template**
4. Copy toàn bộ nội dung từ **STYLING.CSS** vào phần **Styling**
5. Lưu và sử dụng!
